## Requirements

- **Format:** JPG
- **Dimensions:** 400×400px minimum (square aspect ratio)
- **File size:** Under 200KB each (optimise with TinyPNG or similar)

## Animation Sequence

The hero section cycles through these images in this order:
054 → 060 → 064 → 060 → 054 → 057 → 072 → 057 → repeat

Each frame displays for 0.75 seconds (6-second total loop).

## Accessibility

If the user has `prefers-reduced-motion` enabled, the animation is disabled and only 054.jpg displays.
